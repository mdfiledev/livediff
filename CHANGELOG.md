# Changelog

## 0.0.3

- Rename the extension display name from `Live Diff` to `LiveDiff` across the extension UI and Marketplace content.
- Refresh the demo screenshot so the published listing matches the new branding.

## 0.0.2

- Reduce VSIX size by bundling Monaco webview assets instead of shipping the full Monaco package.
- Improve release metadata, icon, and extension details content.
- Refresh the packaged demo screenshot and Marketplace presentation.

## 0.0.1

- Initial release of LiveDiff.
- Open a fresh side-by-side comparison tab from the command palette or shortcut.
- Keep both sides editable while line and inline diffs update live.
- Support multiple comparison tabs at the same time.
